# springclould-eureka

#### Eureka是什么？
云端服务发现，一个基于 REST 的服务，用于定位服务，以实现云端中间层服务发现和故障转移

#### 软件架构
springclould
eureka


#### 服务介绍

1. db-service - 与MySQL数据库进行交互
2. stock-service - 用于从YahooFinance API中提取股票价格
3. eureka-service - 用于注册所有微服务的服务注册中心
